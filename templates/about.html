About Picamera2 WebUI

Lightweight web interface for the Raspberry Pi camera module

Built on the Picamera2 library. This project provides a simple user interface to configure camera settings, capture photos, and manage images in a basic gallery.

Updates on Github

Simply Shoot

Take photos with a single click and save them to the image gallery.

Configure Your Camera

Easily configure camera settings such as image rotation, exposure, white balance settings, and meny more.

Save Your Settings

When you are happy with the configuration of your camera you can save the settings for later, even after rebooting.

Simple Gallery

Veiw, delete, and download your images in a simple gallery interface.

Under The Hood

Picamera2

This project utilizes the Picamera2 library for Python. Picamera2 is the libcamera-based replacement for Picamera which was a Python interface to the Raspberry Pi's legacy camera stack.

Flask is a micro web framework written in Python. It is classified as a microframework because it does not require particular tools or libraries. It has no database abstraction layer, form validation, or any other components where pre-existing third-party libraries provide common functions.
